# 待办事项：项目进度显示功能

## 需求描述
在项目页面上显示当前创作进度，让用户能够清楚地知道：
- 当前已完成多少章节
- 总共需要生成多少章节
- 是否全部完成（显示"已创作完成"或"已编辑完成"）

## 功能要点
1. 在项目列表/详情页显示进度条或进度百分比
2. 显示格式如：`已完成 11/100 章 (11%)`
3. 全部完成时显示状态标签：`✅ 已创作完成`
4. 不影响现有的章节生成流程

## 实现建议
### 后端
- 在 `/api/projects/{project_id}` 返回中添加进度字段
- 计算逻辑：已有内容的章节数 / 总大纲数

### 前端
- 在项目卡片上显示进度条
- 在项目详情页显示详细进度信息
